# JSX, Components and State

## Start

In order to start you will need [NodeJS](https://nodejs.org/en) and [npm](https://www.npmjs.com/) (comes with NodeJS).

1. Get all packages

```shell
cd ./simple-react-app
npm i
```

2. Start dev server

```shell
npm run dev
```

_Note_: It in includes hot module reload.
